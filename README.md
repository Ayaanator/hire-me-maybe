# HireMeMaybe

A satirical job application site that mocks everything wrong with modern hiring.

Browse fake job postings. Fill out ridiculous forms. Answer invasive screening questions. Watch corporate buzzwords fall apart.

It's a love letter to everyone who's applied to a position that was already filled, been rejected by a robot ATS, or had to awkwardly justify their “diverse perspective.”

Built with React and spite.

Website link: https://ayaanator.github.io/hire-me-maybe/
